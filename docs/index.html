>_ Termux-NexusSEC-OS
Ambiente di pentesting su Android · senza root
⚠️ Non installare Termux dal Play Store: quella versione è vecchia e non aggiornabile. Usa una delle fonti ufficiali qui sotto.

1 Scarica Termux

Modo consigliato — F-Droid. Si aggiorna da solo e sceglie in automatico l'APK giusto per il tuo telefono.

🟢 Termux su F-Droidapri e premi "Scarica APK" (richiede l'app F-Droid o il download diretto) CONSIGLIATO

In alternativa — APK ufficiale da GitHub (release v0.118.3). Scegli l'architettura del tuo telefono: quasi tutti i modelli dal 2017 sono arm64-v8a. Se non sei sicuro, usa universal (va bene ovunque, ma pesa di più).

🌍universalva ovunque (~118 MB) 📟armeabi-v7atelefoni vecchi 32-bit 💻x86_64emulatore / tablet Intel 💻x86emulatore 32-bit

Vuoi sempre l'ultima versione? Release più recente su GitHub. Dopo il download, Android chiederà di consentire l'installazione da "origini sconosciute": accetta.

2 Installa NexusSEC-OS

Apri Termux appena installato e incolla questi comandi (tieni premuto per incollare):

pkg update -y && pkg upgrade -y
termux-setup-storage
pkg install -y git python
git clone https://github.com/dPlusOS21/Termux-NexusSEC-OS.git
cd Termux-NexusSEC-OS
bash install.sh
python server.py

Poi apri http://127.0.0.1:8000 nel browser del telefono. install.sh scarica ~700 MB–1 GB (tool + Debian minimale): meglio sotto Wi-Fi.
